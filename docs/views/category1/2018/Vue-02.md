## 基础-过滤器-传参数和串联使用

> **`目标`**掌握在过滤器中如何传参数 和串行使用
>
> - 过滤器可以传递参数,第一个参数永远是前面传递过来的过滤值
> - 过滤器也可以多个串行起来并排使用,
>
> ```js
> // index为传入的参数 
> toUpper(value, index) {
>       return value
>         .split("")
>         .map(function(item, i) {
>           if (i === index) {
>             return item.toUpperCase();
>           }
>           return item;
>         })
>         .join("");
>     }
> } // 根据传入的索引找到对应的字母换成大写字母
> ```
>
> ```html
> <p>{{ text | toUpper(2) }}</p>  
> 
> ```
>
> - 串行使用 过滤器
>
> ```html	
> <p>{{ text | toUpper(2) | reverse }}</p> // 语法 多个过滤器用 | 分割
> 
> ```
>
> **`任务`**:实现一个过滤器,可以根据传去的索引,找到对应的字符转成大写
